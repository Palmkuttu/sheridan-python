# Prime Factors Assignment

## Description
This project implements a function called `generate_prime_factors` that calculates and returns the prime factors of a given positive integer.

The project was developed using **Test Driven Development (TDD)**. Tests were written first, followed by minimal code to make each test pass.

## Function
**generate_prime_factors(n)**

### Parameters:
- `n` (int): A positive integer.

### Returns:
- A list of integers representing the prime factors of `n`.

### Raises:
- `ValueError` if the input is not an integer.
- `ValueError` if the input is less than 1.
